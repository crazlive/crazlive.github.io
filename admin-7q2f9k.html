Khu Vực Quản Trị

Nhập mật khẩu quản trị để truy cập bảng điều khiển CodeHub.vn

</> CodeHub Admin
Cách hoạt động: Trang này không dùng database. Mọi thay đổi được lưu tạm trong trình duyệt (localStorage) để bạn xem trước ngay lập tức trên trang chủ. Khi hoàn tất, bấm “Xuất file index.html” bên dưới để tải file trang chủ đã cập nhật, rồi tải lên (upload) thay thế file cũ trên hosting — thay đổi sẽ hiển thị cho mọi khách truy cập.
Bảo mật: Trang quản trị này có tên file bí mật, không được liên kết ở bất kỳ đâu trên trang chủ. Hãy đổi mật khẩu mặc định (xem hướng dẫn HUONG_DAN_ADMIN.txt), và cân nhắc thêm bảo vệ ở cấp máy chủ (.htaccess / chặn IP) nếu triển khai thật.

📦 Xuất bản thay đổi (Publish)

Bấm nút bên dưới để tải file index.html mới đã bao gồm toàn bộ thay đổi bạn vừa thực hiện. Tải file gốc index.html hiện tại lên cùng thư mục với trang quản trị này trước khi bấm để hệ thống tự động ghép dữ liệu.
